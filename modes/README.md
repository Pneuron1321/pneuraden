## НОТАЦИЯ КОМБИНАЦИЙ

**A × B** = одновременная активация. Мультипликативный эффект.
Пример: `АРХИТИП × OPA` = 5 движений под давлением поляризаций

**A + B** = последовательная активация. Выход A → вход B. Аккумулятивный эффект.
Пример: `ОВЕРХИЛ2+ + МЗ` = найди решение → проверь нашло ли оно форму

**A × B + C** = сначала одновременно A и B, их синтез идёт в C.
Пример: `ОВЕРХИЛ2+ × МС + МЗ` = полный цикл (карта × максимум + завершённость)
